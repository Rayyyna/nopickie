# Release Notes - v1.0.0

**发布日期：** 2025-10-21

## 🎉 首次发布

NoPickie v1.0 正式发布！基于 AI 的实时挠头行为检测应用。

### ✨ 功能列表

- 实时检测挠头/摸脸/揉眼睛等手贱动作
- 智能弹窗提醒（3秒自动消失）
- 调试窗口显示实时检测画面
- 自动截图保存到本地
- 菜单栏快捷控制

### 📦 当前版本

**轻量版：** 需要预装 Python 3.8+ 和依赖包

### 🔜 未来计划

- 完整版（内置 Python，无需安装依赖）
- 更多提醒方式（声音、震动）
- 统计报告功能
- Windows 支持

### 🐛 已知问题

- 菜单栏图标在某些系统版本可能不可见（功能正常）
